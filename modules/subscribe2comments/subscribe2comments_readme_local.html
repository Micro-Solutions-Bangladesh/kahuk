Subscribe To Comments Module

Description

This module gives users an option to follow on articles, by subscribing to email notifications anytime a comment is posted on articles they follow. This feature can be turned ON/OFF by subscribing/unsubscribing.

   

Installation

Install the module from the Dashboard -> Modules -> Uninstalled Modules tab

Settings

To visit the Module's settings, click on the "Subscribe to Comments Module link from the left sidebar under the Modules heading.

The Module has 3 settings, "From Name", "From Email Address" and "Subject Prefix"
You can skip this phase and leave them empty, if you wish to use the default site email and visual name set in the language file, that you normally use to sending welcome and password reset emails, or you can set them up if you wish to use a different email and name. (see picture below)



How to subscribe to comments notifications

When Admins install the module, it inserts a column "auto_comment_alert" with a zero value, in the users table.

Users can subscribe to automatic future comment notifications on all the articles they submit afterwards, by checking the box of "Auto notification to comments on my stories" in their profile settings tab.
Unfortunately, the module does not automatically subscribe the user on all the articles that were submitted prior to installing the module!

Another way for the user to selectively subscribe/unsubscribe, on their articles and other users' as well, is by visiting the article (full view) and click on the "Notify me of new comments" button to subscribe, or "Unssubscribe from comments notification" to unsubscribe.
NOTE that users can also unsubscribe from individual comment notification by clciking the "Unsubscribe" link in the comment email notification they receive.
