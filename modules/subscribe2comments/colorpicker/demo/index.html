Select the background-color for the heading of the comment email notification

Click inside the input field to use the color picker. To test the selected background color on the heading, click outside the box.



After you are satisfied with the background-color, COPY the value in the input field and PASTE it in the Subscribe to Comments Module's settings, under "Heading Background Color"

The font color, that is displayed when you change the background color, is accurate but not 100%! In some instances you have to use your judgment and and decide between TWO colors, Black (#000000) or White (#FFFFFF), and enter the value in the font color field in the module's settings.

The default CSS applied to the comment email notification heading is (background-color:#183a52) and (color:#FFFFFF)
