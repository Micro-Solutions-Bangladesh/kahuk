Links Module v2.0

Description

This module converts URLs in stories and comments to clickable links. You have the option to enable it for comments or stories, and the ability to set the links as 'rel="nofollow"'.

If URLs have (.png;.jpg;.jpeg;.gif) file extension, they will be converted to images

If URLs have (.mp3;.wav;.ogg) file extension, they will be converted to audio player

Revision History

December 2016, January and May 2017. Revised by Redwine and added the following:

  • One of the following settings MUST be checked in order for the module to work; otherwise, you get a blank story description.
  • Allowed to All level users? (If you check this, there is no need to check the below ones)
    is the mostlikely one to check; it allows all users to benefit from converting the URLs into links. If you check it, then there is no need to check the other two!
  • Or just allowed to Moderators?
    You check this one if you just want to allow Moderators to use the Links module!
  • Or just allowed to Admins?
    You check this one if you just want to allow Admins to use the Links module!
  • Make links rel='nofollow' RECOMMANDED
  • Convert Youtube URLs to embedded iframe. Otherwise, converts it to links in Stories
  • Convert Youtube URLs to embedded iframe. Otherwise, converts it to links in comments
  • Convert Facebook URLs to embedded iframe. Otherwise, converts it to links in Stories
  • Convert Facebook URLs to embedded iframe. Otherwise, converts it to links in comments
